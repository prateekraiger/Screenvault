## ✨ Features

### 🎯 **Core Functionality**
- 🎬 **Movies** - Browse and discover cinematic masterpieces

- 🌐 **Web Series** - Digital storytelling at its finest

## 🚀 Quick Start

### 📋 Prerequisites

- **Node.js** v18+ (recommended: v20+)
- **Package Manager**: pnpm (recommended) / npm / yarn

### 📥 Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/screenvault.git
cd screenvault

# Install dependencies
pnpm install
# or
npm install
```

### 🏃‍♂️ Development

```bash
# Start development server
pnpm run dev
# or
npm run dev

# Open http://localhost:5173 in your browser
```

### 🏗️ Production Build

```bash
# Build for production
pnpm run build

# Preview production build
pnpm run preview

# Lint code
pnpm run lint
```

## 📁 Project Structure

```
src/
├── 📂 components/          # Reusable UI components
│   ├── 🎨 ui/             # Base UI components
│   ├── ✨ magicui/        # Enhanced UI with animations
│   ├── 🧭 Navbar.tsx     # Navigation component
│   ├── 🎭 Background.tsx # Animated background
│   └── 🦶 Footer.tsx     # Footer component
├── 📂 pages/              # Page components
│   ├── 🏠 Home.tsx       # Landing page
│   ├── 🎬 MoviesPage.tsx # Movies catalog
│   ├── 📺 TVPage.tsx     # TV shows catalog
│   ├── 🐉 AnimePage.tsx  # Anime catalog
│   ├── 🌐 WebSeriesPage.tsx # Web series catalog
│   ├── 🎵 MusicPage.tsx  # Music integration
│   └── 🎮 GamesPage.tsx  # Games catalog
├── 📂 data/              # Mock data and content
├── 📂 hooks/             # Custom React hooks
├── 📂 lib/               # Utility functions
├── 📂 types/             # TypeScript definitions
└── 🎨 index.css          # Global styles and animations
```

## 🎨 Design System

### 🌈 Color Palette
- **Primary**: Emerald (`#10b981`) → Cyan (`#06b6d4`) → Blue (`#3b82f6`)
- **Background**: Dark slate with gradient overlays
- **Accent**: Teal (`#14b8a6`) for highlights

### ✨ Animation Features
- 🔄 **3D Hover Effects** - Cards rotate and lift on interaction
- 🌊 **Gradient Borders** - Animated rainbow borders on hover
- 💫 **Floating Icons** - Subtle icon animations with pulse effects
- 📝 **Text Gradients** - Dynamic text color transitions
- 🔮 **Glow Effects** - Multi-layered shadow animations
